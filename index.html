📡 Como ativar a sincronização gratuita
  1. Acesse console.firebase.google.com
  2. Crie um projeto gratuito (plano Spark)
  3. Vá em Firestore Database → Criar banco de dados
  4. Em Configurações do projeto → Apps, adicione um app Web
  5. Copie o objeto firebaseConfig e cole no início do script deste arquivo
  6. Em Regras do Firestore, coloque: allow read, write: if true;
  7. Abra o arquivo em qualquer celular e compartilhe o código de 6 letras
Sem o Firebase, a lista funciona normalmente no dispositivo mas sem sincronização.
